# 💰 Daily Expense Tracker (C++)

A robust command-line financial tool developed in C++. This application enables users to manage their daily finances by tracking income and expenditures, with persistent data storage to ensure records are never lost.

![C++](https://img.shields.io/badge/Language-C++-00599C?style=for-the-badge&logo=c%2B%2B)
![Industry](https://img.shields.io/badge/Type-Financial%20Tool-green?style=for-the-badge)

---

### ✨ Key Features
- 📝 **Transaction Logging:** Easily record expense descriptions and amounts.
- 📁 **Persistent Storage:** Utilizes C++ File I/O (`fstream`) to save data in `.txt` format.
- 📊 **Financial Summary:** Calculates and displays total expenditures and remaining balance.
- 🔍 **History Tracking:** Detailed view of all past transactions stored in the database.
